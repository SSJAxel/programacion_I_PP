# 📊 Sistema de Gestión de Legajos

Sistema de consola desarrollado en Python para la gestión y análisis de legajos de alumnos, como proyecto del primer parcial de Programación I — Tecnicatura en Programación, UTN Avellaneda.

Permite cargar datos de alumnos con sus notas, calcular promedios, ordenar registros y consultar información a través de un menú interactivo.

---

## ⚙️ Funcionalidades

- Carga de datos de alumnos por legajo y género
- Cálculo de promedios individuales
- Conteo y clasificación de notas
- Ordenamiento por promedio y por nota mayor
- Visualización de todos los registros cargados
- Menú interactivo por consola
- Código modularizado en múltiples archivos

---

## 🛠️ Tecnologías

- Python 3.x

---

## ▶️ Cómo ejecutarlo

1. Cloná el repositorio:
```bash
git clone https://github.com/SSJAxel/programacion_I_PP.git
cd programacion_I_PP
```

2. Ejecutá el programa principal:
```bash
python parcial.py
```
